# ReDist Outreach Cadence

Date: 2026-06-22  
Owner: Founder  
Status: Founder outreach operating rhythm  
Use with: [ReDist Partnership Pipeline](REDIST_PARTNERSHIP_PIPELINE.md)

## Outreach Principle

The first outreach goal is not scale. The goal is to find one qualified supplier and one qualified recipient for a founder-guided UAE pilot workflow.

Keep every message simple:

- ReDist is a controlled UAE pilot.
- The platform is protected and not publicly launched.
- The ask is a short discovery call or one guided workflow.
- ReDist does not claim live pilot results yet.

## Cadence Overview

| Step | Timing | Goal | Outcome |
| --- | --- | --- | --- |
| First contact | Day 0 | Introduce ReDist and request a 20-minute call | Contacted |
| Follow-up 1 | Day 2 | Re-state low-commitment guidance ask | Reply or no response |
| Follow-up 2 | Day 5 | Close loop and ask for referral | Reply, referral, or no response |
| Demo request | After reply | Book founder-led walkthrough | Meeting booked |
| Pilot invitation | After qualification | Invite to controlled Wave 1 workflow | Invited |

## First Contact

Subject options:

- Controlled UAE pilot for trusted surplus redistribution
- Founder-guided pilot for surplus transfer proof
- Request for guidance on UAE redistribution workflow

Message:

```text
Hi [Name],

I’m the founder of ReDist, a UAE-first platform for verified organizations to redistribute surplus resources with trust, handover verification, certificate-backed evidence, and impact visibility.

We have completed protected staging validation and are now speaking with a small number of UAE organizations before onboarding the first real pilot users.

I would value 20 minutes of your guidance on whether this workflow is useful, risky, or already covered by existing efforts in your sector.

Would you be open to a short call next week?

Regards,
[Founder Name]
```

## Follow-Up 1

Timing: 48 hours after first contact.

```text
Hi [Name],

Just following up on my note about ReDist’s controlled UAE pilot for verified surplus redistribution.

I’m not asking for a commitment yet. I would value 20 minutes of guidance on whether this workflow could help suppliers, recipients, or ESG teams manage surplus handovers with better evidence.

Would a short call next week be possible?

Regards,
[Founder Name]
```

## Follow-Up 2

Timing: 5 days after first contact.

```text
Hi [Name],

I wanted to close the loop on my earlier note.

If ReDist’s controlled pilot is not relevant right now, no problem. If there is someone better to speak with about surplus redistribution, recipient validation, or transfer proof, I would appreciate a pointer.

Thank you,
[Founder Name]
```

## Demo Request

Use after the prospect replies positively.

```text
Thank you, [Name].

The most useful next step would be a 20-minute founder walkthrough:

1. I’ll show the protected pilot workflow.
2. I’ll ask how surplus or recipient needs work in your organization.
3. We’ll identify whether a low-risk pilot use case exists.
4. If there is a fit, we can decide whether to invite your organization into the guided pilot.

Would [two time options] work?
```

## Pilot Invitation

Use only after qualification.

```text
Hi [Name],

Based on our discussion, I’d like to invite [Organization] into ReDist’s founder-guided UAE Pilot Wave 1.

The first workflow will stay narrow:

- One approved organization user.
- One low-risk surplus/resource use case.
- Founder-guided onboarding.
- Request, approval, handover, verification, certificate, and impact review.
- Direct feedback after the workflow.

This is not a public launch. It is a controlled pilot to validate whether ReDist creates practical operational value.

If you’re comfortable proceeding, I’ll schedule onboarding and share the protected staging access instructions privately.

Regards,
[Founder Name]
```

## Segment-Specific First Ask

| Segment | Best First Ask |
| --- | --- |
| NGO / recipient | "Can you review whether this request and certificate workflow would be useful for recipient organizations?" |
| Food redistribution organization | "Can you advise on safe, compliant surplus workflow boundaries?" |
| Hotel group | "Can you identify one low-risk surplus category and an operations owner?" |
| Restaurant group | "Can you introduce one restaurant willing to discuss surplus workflow pain points?" |
| Warehouse operator | "Can you validate whether reusable materials need handover proof and recipient matching?" |
| Corporate ESG team | "Can you review whether certificate-backed redistribution evidence would support ESG reporting?" |

| Daily Action | Target |
| --- | ---: |
| New first-contact messages | 2 |
| Follow-ups | 1-3 |
| Pipeline updates | Every outreach action |
| Meeting preparation | Before every call |

## Weekly Operating Rhythm

Every week:

1. Review pipeline status.
2. Replace no-response targets.
3. Book at least two discovery calls if possible.
4. Select only qualified pilot candidates.
5. Keep product development paused unless a critical pilot blocker appears.
