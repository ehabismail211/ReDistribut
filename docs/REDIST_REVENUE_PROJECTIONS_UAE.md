# ReDist Revenue Projections UAE

Date: 2026-06-21  
Status: Estimate only; real pilot users and paying customers are not yet available  
Currency: AED

## Important Disclaimer

These projections are planning estimates, not forecasts. They should be updated after UAE Pilot Wave 1 produces real data on:

- Willingness to pay.
- Sales cycle length.
- Support load.
- Active usage.
- Certificate/reporting value.
- Conversion from pilot to paid.
- Churn or disengagement.

## Key Revenue Drivers

| Driver | Impact |
| --- | --- |
| Completed transfers | Creates repeat usage and certificate value |
| Certificate trust | Supports willingness to pay |
| ESG reporting demand | Raises ARPA |
| Multi-location use | Drives Professional and Enterprise tiers |
| Founder support load | Limits growth if not systematized |
| Legal/commercial readiness | Determines when paid conversion is possible |

## Key Risks

| Risk | Effect On Revenue |
| --- | --- |
| Organizations like the mission but do not repeat usage | Low conversion |
| Certificate not valued by decision makers | Lower pricing power |
| ESG reports viewed as unvalidated estimates | Add-on revenue delayed |
| Enterprise requires integrations too early | Sales delays |
| Support remains founder-dependent | Growth capped |
| Legal/commercial materials incomplete | Paid launch delayed |
