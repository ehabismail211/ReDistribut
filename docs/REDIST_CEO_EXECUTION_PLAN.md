# ReDist CEO Execution Plan

Status: Active founder operating plan  
Time horizon: First 12 months  
Primary rule: generate real market evidence before adding new product scope

## Phase 1: Market Entry, Next 30 Days

### Primary Objective

Generate the first real market evidence.

### Success Metrics

| Metric | Target |
| --- | ---: |
| Qualified leads | 10 |
| Discovery meetings | 5 |
| Pilot candidates | 2 |
| Supplier commitments | 1 |
| Recipient commitments | 1 |

### Weekly CEO Activities

| Day | Focus | Actions |
| --- | --- | --- |
| Monday | Pipeline control | Review Founder Dashboard, review new leads, update pipeline, set weekly targets. |
| Tuesday | Outreach | Contact 5 organizations, send partnership introductions, follow up existing leads. |
| Wednesday | Discovery | Hold discovery meetings, capture objections, record market insights. |
| Thursday | Conversion review | Review product friction, website analytics, and lead conversion. |
| Friday | CEO review | Update command center, review weekly results, decide next actions. |

### CEO Rule

Spend:

- 80% on outreach and partnerships.
- 20% on product improvements.

Do not reverse this ratio.
