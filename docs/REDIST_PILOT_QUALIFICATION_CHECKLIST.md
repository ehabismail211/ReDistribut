# ReDist Pilot Qualification Checklist

Use: pass/fail checklist before inviting an organization into founder-guided UAE Pilot Wave 1.

## Supplier Qualification

| Requirement | Pass/Fail | Notes |
| --- | --- | --- |
| Has usable surplus |  |  |
| Surplus category is low-risk |  |  |
| Has decision maker |  |  |
| Has coordinator |  |  |
| Can describe quantity and condition |  |  |
| Can define pickup or handover location |  |  |
| Can review recipient requests |  |  |
| Can support one guided pilot workflow |  |  |
| Can provide feedback |  |  |
| No safety, legal, or brand blocker |  |  |

Supplier decision:

- Pass: invite to supplier onboarding.
- Conditional Pass: request missing information.
- Fail: defer or decline for Wave 1.
