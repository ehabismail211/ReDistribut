# ReDist Website Development Sprint Plan

Date: 2026-06-22  
Status: Three-sprint website build plan  
Scope: Marketing website MVP

## Estimated Website Build Effort

Estimated effort: 5-8 focused development days for MVP website, excluding legal review, brand asset production, and advanced CRM integration.

Assumptions:

- Existing web stack can support static marketing pages.
- Contact form uses a simple approved form handler or lightweight backend.
- No product app changes are included.
- No self-serve onboarding is added.

## Sprint 1: Foundation and MVP Pages

Goal:

- Build the basic marketing website structure and most important pages.

Scope:

- Shared layout.
- Header navigation.
- Footer.
- Home page.
- How It Works page.
- Suppliers page.
- Recipients page.
- Reusable Hero, Feature Grid, CTA Block components.

Acceptance:

- Core pages render on desktop and mobile.
- Navigation works.
- CTAs route to Contact with intent query.
- Public claims stay within approved messaging.

Estimated effort:

- 2-3 development days.

## Sprint 2: FAQ, Contact, Lead Capture

Goal:

- Make the website conversion-ready.

Scope:

- FAQ page.
- FAQ accordion or grouped sections.
- Contact page.
- Contact form.
- Form success state.
- Intent query handling.
- Basic SEO metadata.
- Public claim review.

Acceptance:

- FAQ is usable and searchable.
- Contact form captures required fields.
- Test submission works.
- Founder can access submitted lead.
- No sensitive document collection.

Estimated effort:

- 2-3 development days.

## Sprint 3: Analytics, Impact, Polish

Goal:

- Add measurement, optional Impact page, and launch polish.

Scope:

- Analytics tool setup.
- CTA and form event tracking.
- UTM capture.
- Impact page if approved.
- Accessibility pass.
- Performance pass.
- Desktop and mobile founder review.
- Soft-launch checklist.

Acceptance:

- Page views and conversion events tracked.
- Mobile layout passes review.
- Performance is acceptable.
- SEO metadata exists.
- Founder approves soft launch.

Estimated effort:

- 1-2 development days.

## Recommended MVP Website Launch Scope

MVP pages:

- Home.
- How It Works.
- Suppliers.
- Recipients.
- FAQ.
- Contact.

Optional MVP add-on:

- Impact, only if copy clearly states there are no public real pilot results yet.

MVP components:

- Hero.
- Feature grid.
- Workflow timeline.
- Trust indicators.
- FAQ accordion.
- Contact form.
- CTA blocks.

## Post-MVP Backlog

Add after MVP is live and lead quality is understood:

- Partners page.
- About page.
- Press release page.
- Approved case studies.
- Partner logos with written approval.
- Arabic localization.
- CRM integration.

## Launch Recommendation

Build Phase 1 as a soft-launch website for founder outreach, not broad public marketing.

Primary success measure:

- Qualified founder conversations, not raw traffic.
