# ReDist Pricing Strategy

Date: 2026-06-21  
Status: Estimated pricing strategy; not validated by real paying customers yet  
Currency: AED, UAE market planning

## Pricing Recommendation

Recommended launch pricing:

| Tier | Suggested Price | Best For |
| --- | ---: | --- |
| Free Pilot | AED 0 | Founder-approved Wave 1 participants |
| Starter Organization | AED 499 / month | Single-location organizations testing repeat use |
| Professional Organization | AED 1,499 / month | Active suppliers/recipients with regular workflows |
| Enterprise | From AED 6,000 / month | Multi-location groups, ESG teams, government-related entities |

Offer annual pricing at approximately 15-20% discount once contracts and billing operations are ready.

## Tier 1: Free Pilot Tier

Purpose: Validate workflow and gather evidence.

Price: AED 0 during founder-guided pilot.

Included:

- Invite-only pilot access.
- Founder-guided onboarding.
- One organization workspace.
- Limited listings and requests.
- Guided transfer completion.
- Transfer certificate review.
- Basic impact review.
- Feedback capture.

Suggested limits:

| Limit | Value |
| --- | ---: |
| Organizations | 1 |
| Users | 2 |
| Listings | 3 |
| Requests | 5 |
| Completed transfers | 2 |
| Certificates | 2 |
| Support | Founder-guided only |

Upgrade trigger:

- Organization wants repeat workflows.
- Organization asks to invite more users.
- Organization wants more listings, requests, or certificates.
- Organization wants reporting beyond pilot review.

## Tier 2: Starter Organization

Purpose: First paid plan for single-location organizations.

Suggested UAE pricing: AED 499 / month or AED 4,999 / year.

Included:

- One verified organization workspace.
- Up to 5 users.
- Up to 20 active listings per month.
- Up to 40 requests per month.
- Up to 20 completed transfers per month.
- Standard transfer certificates.
- Basic impact dashboard.
- Email support.
- Basic organization verification workflow.

Suggested limits:

| Limit | Value |
| --- | ---: |
| Locations | 1 |
| Users | 5 |
| Active listings/month | 20 |
| Requests/month | 40 |
| Certificates/month | 20 |
| ESG reports | Not included |
| Support | Standard |

Upgrade path:

- More than one location.
- More than 20 certificates/month.
- Need executive ESG reporting.
- Need priority support.
- Need multiple departments or branches.

## Tier 3: Professional Organization

Purpose: Default plan for active supplier or recipient organizations.

Suggested UAE pricing: AED 1,499 / month or AED 14,999 / year.

Included:

- One verified organization workspace.
- Up to 3 locations or branches.
- Up to 20 users.
- Up to 100 active listings per month.
- Up to 200 requests per month.
- Up to 100 certificates per month.
- Advanced impact dashboard.
- Certificate history.
- Priority support.
- Quarterly pilot/business review.
- CSV/manual reporting export when available.

Suggested limits:

| Limit | Value |
| --- | ---: |
| Locations | 3 |
| Users | 20 |
| Active listings/month | 100 |
| Requests/month | 200 |
| Certificates/month | 100 |
| ESG reports | Basic quarterly summary |
| Support | Priority |

Upgrade path:

- More than 3 branches.
- Need group-level reporting.
- Need custom verification/reporting workflows.
- Need dedicated account management.
- Need SSO, API, integrations, or custom contract terms.

## Tier 4: Enterprise

Purpose: Large organizations and multi-location groups.

Suggested UAE pricing: From AED 6,000 / month or AED 60,000 / year.

Enterprise pricing should vary by:

- Number of branches.
- Number of users.
- Transfer/certificate volume.
- Reporting requirements.
- Support requirements.
- Integration needs.
- Security/legal review complexity.

Included:

- Multi-location organization setup.
- Custom user and branch onboarding.
- Higher listing/request/certificate volume.
- Executive impact reporting package.
- Certificate evidence library.
- Dedicated success support.
- Priority verification review.
- Quarterly business review.
- Optional paid implementation support.
- Integration discovery.

Suggested starting limits:

| Limit | Value |
| --- | ---: |
| Locations | 10 included, more quoted |
| Users | 50 included, more quoted |
| Active listings/month | 500 |
| Requests/month | 1,000 |
| Certificates/month | 500 |
| ESG reports | Executive monthly/quarterly |
| Support | Dedicated success contact |

Upgrade / expansion path:

- Additional locations.
- Additional certificate volume.
- ESG report customization.
- API/inventory integration.
- Government or audit support package.

## Upgrade Path

Free Pilot -> Starter:

- First transfer completed.
- Organization wants repeat use.
- Organization accepts basic pricing.

Starter -> Professional:

- More users, volume, requests, certificates, or reporting.
- Organization sees recurring value.

Professional -> Enterprise:

- Multi-location needs.
- Executive ESG reporting.
- Governance, support, or integration requirements.

## Pricing Caveats

- Pricing is estimated and should be validated in sales conversations.
- Do not publish pricing publicly until founder approves positioning.
- Do not sell audited ESG claims unless methodology and legal review support it.
- Do not introduce payment processing until billing operations and terms are ready.
- Manual invoicing is acceptable for first paid customers.

## Recommended First Paid Offer

Offer first paying customer a 90-day Professional pilot conversion:

- AED 1,499 / month.
- Founder-guided onboarding included.
- Up to 3 locations.
- Up to 100 certificates/month.
- Quarterly impact review.
- Month-to-month with conversion to annual after value proof.
