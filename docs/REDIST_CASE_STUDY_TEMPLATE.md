## Value Recovered

Value recovered should be stated carefully.

Allowed:

- "Estimated AED value recovered."
- "Supplier-provided estimated value."
- "Replacement value estimate."

Avoid:

- Audited value.
- Guaranteed value.
- Tax-deductible value.
- Financial benefit unless verified.

Draft:

```text
The transfer represented an estimated AED [amount] in recovered value, based on [method]. This estimate is unaudited and provided for pilot reporting context.
```
