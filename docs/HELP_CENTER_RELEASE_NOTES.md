# Help Center Release Notes

Date: 2026-06-22  
Status: Ready for validation  
Scope: Public and platform documentation access

## Summary

ReDist now exposes bilingual user documentation through a public Help Center and an internal workspace Help Center.

## Public Help Center

URL: `/help`

Includes:

- Quick Start Guide
- User Manual
- Supplier Guide
- Recipient Guide
- FAQ preview and link to the full FAQ
- English and Arabic instructions
- Real ReDist screenshots from the bilingual manual package

## Platform Help Center

URL: `/app/help`

Includes the same bilingual guide sections as the public Help Center, plus founder support instructions for pilot participants.

## Navigation Updates

Public website:

- Added Help Center to primary and footer navigation.
- FAQ remains directly available.

Workspace:

- Added Support navigation group.
- Added Help Center link.
- Added FAQ link.

## Documentation Coverage

Covered workflows:

- Getting started
- Dashboard
- Discover resources
- Supplier listing workflow
- Recipient request workflow
- Requests and transfers
- Certificates
- Impact
- Founder support escalation
