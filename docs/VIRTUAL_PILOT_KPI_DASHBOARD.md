## KPI Recommendation

Use the virtual pilot dashboard to guide the first real pilot:

- Start with one non-food transfer.
- Use Supplier B or Supplier C style workflows first.
- Keep food-adjacent workflows gated until compliance language and handling rules are reviewed.
- Review certificate wording with every participant before claiming impact.
