## Scoring Model

Score each category from 1 to 5.

| Score | Meaning |
| ---: | --- |
| 1 | Weak fit / unclear / high friction |
| 2 | Some relevance but significant concerns |
| 3 | Possible fit with founder guidance |
| 4 | Strong fit with manageable friction |
| 5 | Excellent fit and ready for pilot action |

| Total Score | Recommendation | Action |
