# ReDist Target Organization List

Date: 2026-06-22  
Owner: Founder  
Status: Target list for UAE outreach and partner discovery  
Source note: Built from existing ReDist pilot outreach materials. Confirm contacts manually before outreach.

## Targeting Principle

Start with organizations that can either:

- Supply usable surplus.
- Receive useful resources.
- Introduce qualified suppliers or recipients.
- Validate trust, compliance, certificate, or ESG reporting needs.

Do not start with broad public launch targets. Start with controlled, founder-guided conversations.

## Founder Targeting Notes

- Keep first pilot workflows narrow and low-risk.
- Use certificate and impact evidence as discussion topics, not proof of completed outcomes.
