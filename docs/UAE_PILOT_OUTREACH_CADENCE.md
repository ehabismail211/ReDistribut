## No-Code Rule

Do not start new product work unless one of these happens:

- Live staging validation fails.
- A security issue is discovered.
- A real pilot prospect identifies a blocker that prevents onboarding.
- A real pilot user fails a core workflow.

Everything else goes into the post-pilot backlog.
